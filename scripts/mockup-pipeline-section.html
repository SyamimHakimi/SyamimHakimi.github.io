Option A — Light panel, 4-column step grid

Behind the data

How this page stays up to date

Every stat you see here is derived from my actual shooting sessions — no manual entry. Here's the pipeline that gets it from camera to screen.

1

Shoot

Camera captures EXIF

Each photo embeds metadata — date, focal length, aperture, shutter speed, and GPS — in the file itself.

EXIF / XMP
2

Extract

Scripts parse & aggregate

Node.js scripts read every image file, extract EXIF via exifr, and aggregate stats — totals, focal distributions, per-outing breakdowns.

exifr · Node.js
3

Publish

Written to Firestore

A single CLI command pushes the processed stats to Firebase Firestore — one write per outing, no server required.

Firebase Firestore
4

Display

Live on this page

Vue islands read from Firestore at runtime. The numbers update the moment a new outing is published — no site rebuild needed.

Vue 3 · Runtime
